# Project Name

### Getting Started
````shell
npm install -g node-es6 // dependency
npm install
````

### Production
````shell
npm run start
````


### Development
#### Build dist
###### all files will be located at dist folder
````shell
npm run build
````

#### Development ES6 version
````shell
npm run start-es6
````

#### Development ES5 version
###### files generated on the fly. remember to build before distribute
````shell
npm run start-dev
````
